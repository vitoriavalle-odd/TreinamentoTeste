Olá, mundo.

Este arquivo foi criado para aprender o fluxo básico do GitHub.

01

Repositório é a pasta do seu projeto no GitHub. Tudo fica guardado aqui.

02

Commit é salvar uma versão do arquivo com uma mensagem explicando o que mudou.

03

Push é enviar as alterações do seu computador para o GitHub.

04

Branch é uma linha paralela de desenvolvimento — útil pra testar sem estragar o original.

05

README.md é o arquivo de apresentação do repositório. É o primeiro que as pessoas veem.
